Parser --- Printf and Scanf for columns
-------------------------------------
Installation.
1)Go to the folder with program. 
2)Type make to make Parser. 
3)Then run program by typing ./parser in your terminal
-------------------------------------
Using program.
Follow the menu you'll see after running the program.
-------------------------------------
You can find the example of input file in Example.txt
